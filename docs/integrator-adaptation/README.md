# Entegrator Uyarlama Dokumanlari

Bu klasor, entegrator firma ile yapilacak teknik uyarlama dokumanlarini toplamak icin acilmistir.

## Onerilen Icerik

1. API endpoint listesi (e-Fatura, e-Irsaliye, e-Arsiv).
2. Kimlik dogrulama ve yetkilendirme akisi.
3. Dokuman cekim kriterleri (tarih araligi, durum, sayfalama).
4. Hata kodlari ve yeniden deneme stratejisi.
5. Ornek request/response payloadlari.
6. SLA, zaman asimi ve oran limitleri.
7. Ortam bilgileri (test, pre-prod, prod).
